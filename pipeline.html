⚙ Configuración — Sincronización Cloud

1. Ve a console.firebase.google.com e inicia sesión con tu cuenta Google.
2. Crea un proyecto (ej: ningenia-pipeline), activa Firestore Database en modo producción.
3. En Configuración del proyecto → Tus apps → añade app Web (</>). Copia los datos de configuración.
4. En Firestore → Reglas, pega esto para permitir acceso:
rules_version='2'; service cloud.firestore { match /databases/{db}/documents { match /{document=**} { allow read, write: if true; } } }
